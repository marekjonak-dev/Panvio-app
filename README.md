# Panvio – Fresh Food Marketplace App

Panvio is a people-to-people platform for buying and selling fresh food locally. It focuses on making local food exchange simple and accessible.

---

## 🚀 Key Features

- Product search for local buyers
- Map search for products and sellers  
- Ordering system  
- Real-time messaging between users  
- QR code verification  
- Multilingual support  
- Basic analytics
- User authentication
- Buyer/Seller accounts
- Seller subscription

---

## 🛠️ Tech Stack

- Flutter / Dart (mobile app)
- Backend: Firebase
- Database: Cloud Firestore
